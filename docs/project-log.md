## 2026-06-14 — Помощь: ошибка на вопрос «какие дома есть?»

- Причина: RAG/ответ занимали слишком долго → `TimedOut` при отправке в Telegram; «споткнулся».
- Быстрый ответ на общие вопросы о домах без RAG → кнопка «🏠 Подобрать дом».
- При «❓ Помощь» сбрасываются зависшие диалоги; RAG прогревается при старте; таймауты Telegram 30 с.

**Проверка:** Помощь → «какие дома у вас есть?» → ответ с числом домов и меню.

## 2026-06-14 — Обратный звонок: телефон + время → группа менеджеров

- Ошибка `tel:` в inline-кнопке: Telegram Bot API не принимает `tel:` — кнопку убрали, номер в тексте.
- Сценарий: номер → удобное время → сообщение в группу `MANAGER_CHAT_ID` (по умолчанию `-1003646706475`).
- Модуль `notifications.py`, переменная `MANAGER_CHAT_ID` в `.env`.

**Проверка:** подбор → «📞 Заказать обратный звонок» → телефон → время → сообщение в группе.
