# Материалы для сдачи выпускного проекта

**Не удалять** — здесь образец формата сдачи от Zerocoder.

| Файл | Назначение |
|------|------------|
| `КП. Пример. ТГ-бот.pdf` | **Образец коммерческого предложения** — структура, разделы, скриншоты |
| `../zerocoder-assignment.txt` | Полный текст задания курса (ТЗ, критерии, что сдать) |

## Что сдать на платформе

1. **КП (PDF)** — по образцу `КП. Пример. ТГ-бот.pdf`, но про **ваш** бот аренды домов
2. **Отчёт** — текст в поле домашнего задания
3. **GitHub** — https://github.com/Ollegro/rental-ai-assistant-graduation
4. **Резюме и портфолио**

## Структура КП (из образца)

1. Описание проекта (цель, технологии, деплой)
2. Основные функции бота
3. Реализация (интервью → ТЗ → разработка)
4. Внедрение и передача
5. Сроки, стоимость, условия
6. **Аналогичный кейс** — скриншоты работы бота

Свой КП делайте по этому шаблону: замените тему «подписки» на «аренда домов + RAG».
